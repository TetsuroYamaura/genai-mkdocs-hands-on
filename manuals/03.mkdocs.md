### PDFへ変換

mkdocs.ymlへ、下記のto-pdfブロックを追記する。

```yml
plugins:
  - search
  - awesome-pages
  - to-pdf:
      cover_subtitle: '生成AI時代のドキュメント基盤'
      toc_title: 目次
      toc_level: 3
      output_path: pdf/生成AI時代のドキュメント基盤.pdf
      enabled_if_env: ENABLE_PDF
  - table-reader:
      enabled: true
```

PDFをビルドする。

```bash
ENABLE_PDF=1 uv run mkdocs build
```

#### MermaidのPDF出力対応

mkdocs.ymlへ、下記のmermaid-to-svgブロックを追記する。

```yml
plugins:
  - search
  - awesome-pages
  - mermaid-to-svg:
      enabled_if_env: RENDER_SVG
      image_id_enabled: true
      image_id_prefix: mermaid-diagram
      # Disable HTML labels for PDF compatibility
      mermaid_config:
        htmlLabels: false
        flowchart:
          htmlLabels: false
        class:
          htmlLabels: false
  - svg-to-png:
      enabled_if_env: RENDER_PNG
  - to-pdf:
      cover_subtitle: '生成AI時代のドキュメント基盤'
      toc_title: 目次
      toc_level: 3
      output_path: pdf/生成AI時代のドキュメント基盤.pdf
      enabled_if_env: ENABLE_PDF
```

PDFをビルドする。

```bash
RENDER_SVG=1 ENABLE_PDF=1 uv run mkdocs build
```
