# textlintを活用する

下記の2ファイルをルートディレクトリにコピーする

- /reference/04.textlint/.textlintignore
- /reference/04.textlint/.textlintrc.json

```bash
cp -rf reference/04.textlint/. .
```

textlintの除外設定と、textlintの設定ファイルになります。

コマンドラインから、つぎのように実行してエラーをチェックできます。

```bash
nr lint:text
```

.vscode/settings.jsonに、下記のtextlintで始まる設定を追加することで、VS Code上で編集中に確認ができるようになります。

```json
{
  "textlint.configPath": ".textlintrc.json",
  "textlint.autoFixOnSave": false,
  "textlint.run": "onType",
  "chat.useAgentsMdFile": false,
```

また定型的に対処可能なエラーは、つぎのように実行して修正が可能です。

```bash
nr lint:text:fix
```
